## 1. eSOP电子作业指导书

### 1.1 功能定位

eSOP将传统纸质作业指导书转换为数字化形式，推送至工位终端显示，实现无纸化车间管理。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 文档管理 | SOP文档上传、版本管理 |
| 工位推送 | SOP推送至工位终端 |
| 阅读确认 | 电子签名确认阅读 |
| 报工拦截 | 未确认SOP阻止报工 |

### 1.3 页面路由

| 页面 | 路由 | 说明 |
|------|------|------|
| 文档管理 | /production/sop/document-list | SOP文档列表 |
| 工位终端 | /production/sop/workstation/:workstationId | 工位终端视图 |
| 签名记录 | /production/sop/signature-records | 签名记录查询 |

### 1.4 业务流程

```
上传SOP文档 → 审批发布 → 工单开工自动推送 → 员工阅读确认 → 可报工
```

## 2. BPM流程引擎

### 2.1 功能定位

BPM流程引擎支持业务流程的定义、执行、审批、监控，提供图形化流程设计和表单配置。

### 2.2 核心功能

| 功能 | 说明 |
|------|------|
| 流程模型管理 | 流程定义、发布、版本管理 |
| 表单设计 | 可视化表单配置 |
| 流程实例 | 发起、审批、监控 |
| 任务分配 | 动态任务分配规则 |

### 2.3 页面路由

| 页面 | 路由 | 说明 |
|------|------|------|
| 流程模型管理 | /bpm/model-list | 流程模型列表 |
| 流程设计器 | /bpm/model-designer/:id | 图形化流程设计 |
| 表单设计器 | /bpm/form-designer/:id | 表单设计 |
| 我的申请 | /bpm/my-applications | 我发起的流程 |
| 我的待办 | /bpm/my-tasks | 待审批任务 |

### 2.4 流程节点类型

| 节点类型 | 图标 | 说明 |
|---------|------|------|
| START | 绿色圆 | 流程开始 |
| END | 红色圆 | 流程结束 |
| APPROVAL | 蓝色方框 | 审批节点 |
| COPY | 黄色菱形 | 抄送节点 |
| CONDITION | 绿色菱形 | 条件分支 |
| ACTION | 紫色方框 | 系统自动执行 |

### 2.5 审批规则

| 规则类型 | 说明 |
|---------|------|
| 会签(ALL) | 所有审批人必须全部同意 |
| 或签(ANY) | 任意审批人同意即可通过 |
| 顺序审批 | 按配置顺序依次审批 |

## 3. 器具容器管理

### 3.1 功能定位

器具容器管理用于管理周转箱、托盘、料盒、工装等容器设备，支持全生命周期管理。

### 3.2 核心功能

| 功能 | 说明 |
|------|------|
| 容器台账 | 容器列表、状态管理 |
| 容器绑定 | 容器与物料绑定 |
| 初始化管理 | 新容器登记检验 |
| 维修管理 | 维修记录管理 |
| 借用归还 | 借用/归还操作 |
| 报废管理 | 容器报废审批 |

### 3.3 页面路由

| 页面 | 路由 | 说明 |
|------|------|------|
| 容器台账 | /wms/container-list | 容器列表 |
| 容器绑定 | /wms/container-binding | 物料绑定 |
| 容器初始化 | /wms/container-init | 初始化登记 |
| 容器维修 | /wms/container-maintenance | 维修记录 |
| 容器借用 | /wms/container-borrow | 借用归还 |
| 容器报废 | /wms/container-scrapped | 报废管理 |

### 3.4 容器状态流转

```
ACTIVE ──[借用]──► IN_USE ──[归还]──► ACTIVE
   │                        │
   ▼                  [损坏/逾期]
MAINTENANCE ──[修复]──► ACTIVE
   │
   ▼
SCRAPPED
```

## 4. AI质检模块

### 4.1 功能定位

AI质检模块与视觉检测系统对接，实现外观缺陷自动检测和质量预测。

### 4.2 核心功能

| 功能 | 说明 |
|------|------|
| 视觉检测对接 | 接收视觉系统检测结果 |
| AI结果查询 | 检测结果列表 |
| 人工确认 | AI判断结果人工复核 |

### 4.3 API接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /vision/results | 接收视觉检测结果 |
| GET | /vision/results | 检测结果列表 |
| PUT | /vision/results/:id/confirm | 人工确认 |
