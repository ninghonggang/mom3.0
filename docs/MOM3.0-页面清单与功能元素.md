## 三、常见页面模式总结

### 列表页面标准元素
大多数列表页面包含：
- **查询区域**: 多个输入框 + 查询/重置按钮
- **操作按钮**: 新增（新建）、编辑、删除
- **表格**: 显示数据列表，支持分页
- **新建/发送**: 创建新记录

### 表单弹窗操作
- 新增/编辑 → 弹出表单对话框
- 保存后自动刷新列表

### 页面功能分类
1. **基础CRUD**: 系统管理、大部分主数据
2. **业务执行**: 生产工单、采购订单、销售订单
3. **检验类**: IQC/IPQC/FQC/OQC
4. **分析报表**: SPC、MPS/MRP计算
5. **待开发**: BPM、AGV、EAM、Fin、Report等
