  ```go
  type MesSop struct {
      ID              uint64    `gorm:"primaryKey"`
      TenantID        int64     `gorm:"index"`
      SopName         string    `gorm:"size:200"`      // SOP文档名称
      FileName        string    `gorm:"size:500"`      // 文件名

      Uploader        string    `gorm:"size:100"`      // 上传人

      UpdateTime      time.Time
  }
  ```

  - GET `/mes/sop/:id` - 获取SOP详情
  - DELETE `/mes/sop/:id` - 删除SOP文档

## 五、技术备注

### SOP文件存储
- 文件存储到 `uploads/sop/` 目录
- 文件名格式: `{tenantID}_{timestamp}_{originalname}`

### 报工状态机
```
待报工(READYTODO) → 报工中(PROCESSING) → 已完工(FINISHED)
```

### 库区层级
```
仓库(Warehouse, L=1) → 库区(Area, L=2) → 库位(Location, L=3)
```
