# MOM3.0 E2E 测试计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为 MOM3.0 项目建立完整的端到端测试体系，覆盖所有核心模块的 CRUD 操作和跨模块业务流程。

**Architecture:** 
- 测试框架: Playwright + TypeScript
- 测试辅助: 数据库直接操作 (db helper)
- 测试分层: 登录测试 → CRUD测试 → 业务流程测试 → 冒烟测试

**Tech Stack:** 
- Playwright 1.58+
- PostgreSQL (pg driver)
- Vue3 + Element Plus

**项目规模:**
- 120+ 路由
- 18 个核心模块
- 20+ 后端 Handler

  supplierId: number
  expectedDate?: string
}): Promise<number> {
  const sql = `
    INSERT INTO scp_purchase_order (tenant_id, order_no, supplier_id, expected_date, status, created_at)
    VALUES (1, $1, $2, $3, 1, NOW())
    RETURNING id
  `

    data.orderNo,
    data.supplierId,

  ])
  return Number(result[0].id)
}

  })

  test('未登录访问受保护路由重定向', async ({ page }) => {
    await page.goto('/dashboard')
    await page.waitForURL('**/login**', { timeout: 5000 })
    await expect(page.url()).toContain('/login')
  })

  test('控制台无关键错误', async ({ page }) => {
    const errors: string[] = []
    page.on('console', msg => {
      if (msg.type() === 'error') {
        errors.push(msg.text())
      }
    })
    await page.goto('/login')
    await page.waitForTimeout(2000)
    const criticalErrors = errors.filter(e =>
      !e.includes('favicon') &&
      !e.includes('WebSocket') &&
      !e.includes('ws://') &&
      !e.includes('net::ERR')
    )
    expect(criticalErrors).toHaveLength(0)
  })
})
