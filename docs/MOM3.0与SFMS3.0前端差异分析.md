| 基础数据-供应商 | 4+ | 供应商/供应商物料/周期/价格 |
| 基础数据-标签管理 | 8+ | 标签类型/模板/条码规则/叫料 |
| 基础数据-策略设置 | 25+ | 上架/下架/批策略/库容策略 |
| 基础数据-单据设置 | 8+ | 业务类型/编号规则/开关 |
| 库存管理 | 10+ | 台账/变化历史/容器/盘点 |

### 2.2 MES执行模块

**SFMS3.0 MES (30+页面):**
- 生产日计划、工单排程、工艺路线、BOM管理
- 工位/班组/假期/能力档案、工序类型/管理
- 产品档案、物料申请、拆解管理、返工批次
- 月计划、花字品种、工作日历、质量表单
- 产品报交/下线/返线、排程详情

**MOM3.0 MES (5页面):**
- 工艺路线 /mes/process-route
- 班组管理 /mes/team
- 人员技能 /mes/person-skill
- 物料追溯 /mes/material-trace
- 下线管理 /mes/offline

**差距**: MOM3.0 MES功能大幅简化，缺少日计划、排程、BOM等核心功能。

---

### 2.3 BPM流程模块

**SFMS3.0 BPM (30+页面):**
- 流程模型设计器 (BPMN)
- 模型/定义/表单管理
- 用户组、任务分配规则
- 流程实例、待办/已办任务
- 任务转派、退回、审批
- OA请假示例

**MOM3.0 BPM (3页面):**
- 流程定义 /bpm/process
- 流程实例 /bpm/instance
- 任务列表 /bpm/task

**差距**: MOM3.0缺少流程设计器、表单设计器、任务分配规则等。

**MOM3.0 SYSTEM (13页面):**

## 4. 结论

MOM3.0前端设计文档相比SFMS3.0存在较大差距，主要体现在:

1. **WMS模块**: MOM3.0仅9页，SFMS3.0有300+页，缺失约290页
2. **MES模块**: MOM3.0仅5页，SFMS3.0有30+页，缺失约25页
3. **BPM模块**: MOM3.0仅3页，SFMS3.0有30+页，缺失约27页
4. **INFRA模块**: MOM3.0完全缺失，SFMS3.0有12+页

**建议**: 如果MOM3.0需要完整功能，应按优先级逐步补充WMS、MES、BPM等模块的详细页面设计。
