# MOM3.0 报表模块设计文档

**版本**: V2.0 | **所属模块**: M15报表 | **基于**: [MOM3.0_主设计文档](./MOM3.0_主设计文档.md)

---

## 1. 模块概述

### 1.1 功能定位

报表平台提供标准报表和自定义报表两类能力，支持生产、质量、设备、库存等全方位数据分析与决策支持。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 生产日报 | 日生产统计数据 |
| 质量周报 | 周质量汇总分析 |
| 设备OEE | 设备综合效率报表 |
| 交货准时率 | 交付率分析 |
| 不良帕累托 | 不良原因分析 |
| 车间大屏 | 生产指挥可视化 |
| 自定义报表 | 低代码报表配置 |
