# MOM3.0 追溯与数据采集模块设计文档

**版本**: V2.0 | **所属模块**: M08数据采集、M10追溯管理 | **基于**: [MOM3.0_主设计文档](./MOM3.0_主设计文档.md)

---

## 1. 模块概述

### 1.1 功能定位

追溯模块实现产品从原材料到成品的全流程追溯，数据采集模块提供实时数据采集和记录。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 序列号追溯 | 产品序列号追溯查询 |
| 批次追溯 | 批次追溯查询 |
| 工单追溯 | 生产过程追溯 |
| 前向追溯 | 投料追溯 |
| 后向追溯 | 成品追溯 |
| 数据点管理 | 采集点配置 |
| 采集记录 | 实时数据存储 |

## 4. 业务流程

### 4.1 追溯链路

```
原材料追溯: 成品 → 工序 → 投料 → 原材料批次 → 供应商
生产追溯: 成品 → 工单 → 工艺参数 → 设备参数
```

---

## 5. 数据模型

### 5.1 追溯记录

| 字段 | 类型 | 说明 |
|------|------|------|
| trace_no | VARCHAR(64) | 追溯编号 |
| serial_no | VARCHAR(100) | 序列号 |
| batch_no | VARCHAR(100) | 批次号 |
| product_id | BIGINT | 产品ID |
| production_date | DATE | 生产日期 |

### 5.2 原材料追溯

| 字段 | 类型 | 说明 |
|------|------|------|
| trace_no | VARCHAR(64) | 追溯编号 |
| material_id | BIGINT | 物料ID |
| material_batch | VARCHAR(100) | 物料批次 |
| supplier_id | BIGINT | 供应商ID |
| qty_used | DECIMAL(18,3) | 用量 |

### 5.3 工序追溯

| 字段 | 类型 | 说明 |
|------|------|------|
| trace_no | VARCHAR(64) | 追溯编号 |
| process_seq | INTEGER | 工序序号 |
| process_name | VARCHAR(100) | 工序名称 |
| workstation_id | BIGINT | 工位ID |
| operator_name | VARCHAR(50) | 操作员 |
| start_time | TIMESTAMP | 开始时间 |
| end_time | TIMESTAMP | 结束时间 |

### 5.4 数据点

| 字段 | 类型 | 说明 |
|------|------|------|
| point_code | VARCHAR(50) | 点位编码 |
| point_name | VARCHAR(100) | 点位名称 |
| data_type | VARCHAR(20) | 数据类型 |
| unit | VARCHAR(20) | 单位 |
|采集频率 | INTEGER | 采集频率(秒) |

---

## 6. API接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /trace/forward/:serialNo | 正向追溯(产品→原材料) |
| GET | /trace/backward/:batchNo | 反向追溯(原材料→产品) |
| GET | /trace/detail/:traceNo | 追溯详情 |
| GET | /data/points | 数据点列表 |
| POST | /data/points | 创建数据点 |
| GET | /data/collection-log | 采集记录 |
| POST | /data/collect | 采集数据上报 |
