## 二、通过通用接口配置平台可替代

以下设计项可通过 M14 通用接口配置平台（已实现）配置化实现，无需硬编码：

| 设计项 | 替代方案 | 可行性 |
|--------|---------|--------|
| 金蝶BOM同步 | 配置 InterfaceConfig，SourceType=TABLE_QUERY，Outbound到金蝶API | ✅ 可行 |
| 报工回传ERP | 配置 InterfaceConfig，TriggerType=EVENT(PRODUCTION_COMPLETE) | ✅ 可行 |
| 入库通知ERP | 配置 InterfaceConfig，TriggerType=EVENT(STOCK_IN) | ✅ 可行 |
| AGV任务下发 | 配置 InterfaceConfig，TriggerType=EVENT(PRODUCTION_COMPLETE) | ✅ 可行 |
| 消息推送飞书 | 配置 InterfaceConfig，TriggerType=SCHEDULE/DYNAMIC，WebHook地址 | ✅ 可行 |
| 供应商Portal API | 配置 InterfaceConfig，Direction=OUTBOUND，认证配置 | ✅ 可行 |
