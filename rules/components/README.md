# 组件规范

本目录定义 MOM3.0 项目中公共组件的使用规范和最佳实践。

## 文件清单

- **[form-components.md](./form-components.md)** - 表单组件规范
- **[table-components.md](./table-components.md)** - 表格组件规范
- **[chart-components.md](./chart-components.md)** - 图表组件规范

## 组件分类

### 基础组件 (Basic)
- 按钮、输入框、下拉框等 Element Plus 基础组件的封装
- 统一的样式和行为

### 业务组件 (Business)
- 特定业务场景的复用组件
- 如：用户选择器、物料选择器、设备状态指示

### 图表组件 (Chart)
- ECharts 图表封装
- 统一的配置和交互模式
